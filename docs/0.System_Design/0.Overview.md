## The Multi-App Ecosystem

### Rider Experience
- **Search**: Real-time destination discovery via Google Places.
- **Track**: Live driver movement on a smooth map.
- **Pay**: Integrated Razorpay checkout with offer code support.

### Driver Experience
- **Offers**: Instant pickup alerts with audio-pings.
- **Navigate**: Turn-by-turn route visualization.
- **Earnings**: Real-time wallet tracking and instant withdrawals.

### Admin Operations
- **God's Eye View**: Real-time map of every active trip and driver.
- **Control**: Dynamic surge pricing and fare configuration.
- **Trust**: Manual verification queue for driver documentation.
