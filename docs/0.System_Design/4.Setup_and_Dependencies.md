# Setup and Dependencies

This guide covers preparing the environment, installing dependencies, and launching the platform.

## Architecture Stack

### Backend (Django REST)
- Core: Python 3.11, PostgreSQL
- Real-time: Django Channels, Redis
- Tasks: Celery, Flower
- Integration: Razorpay, Google Maps, Expo SDK

### Frontend (Web)
- Framework: React 18, Vite, TypeScript
- State: Zustand
- Maps: Google Maps JS API

### Mobile (React Native)
- Platform: Expo SDK
- Mapping: React Native Maps
- Tracking: Expo Location

## Docker Infrastructure

### Core Commands
| Action | Command |
| :--- | :--- |
| Start Stack | `docker compose up -d` |
| Rebuild | `docker compose up -d --build` |
| Stop All | `docker compose down` |
| Clear Data | `docker compose down -v` |

### Log Monitoring
```bash
# View all logs
docker compose logs -f

# Specific Service Logs
docker logs -f uber_backend
docker logs -f uber_celery
docker logs -f uber_kafka
```

## Running the Applications

### 1. Backend Management
```bash
# Apply Database Migrations
docker exec -it uber_backend python manage.py migrate

# Create Superuser
docker exec -it uber_backend python manage.py createsuperuser

# Access Shell
docker exec -it uber_backend python manage.py shell

#TEST
docker exec uber_backend pytest -v
```

### 2. Web Dashboards (React)
- **Admin Dashboard**:
  ```bash
  cd admin-dashboard
  npm install
  npm run dev
  ```
- **Rider Web**:
  ```bash
  cd frontend/rider-web
  npm install
  npm run dev
  ```

### 3. Mobile Applications (Expo)
- **Driver App**:
  ```bash
  cd frontend/driver-app
  npm install
  npx expo start
  ```
- **Rider App**:
  ```bash
  cd frontend/rider-app
  npm install
  npx expo start --dev-client
  ```



## Unified Startup Sequence

1. Populate `.env` with API keys.
2. Run `docker compose up -d`.
3. Run `python manage.py migrate` via Docker.
4. Verify health at `http://localhost:8000/api/common/health/`.
5. Launch the required frontend dashboard or mobile app.
